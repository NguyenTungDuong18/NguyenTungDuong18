- Phát triển các ứng dụng Web đáp ứng chính xác và hiệu quả các yêu cầu nghiệp vụ thực tế.

- Hoàn thành tốt chương trình học tại đại học và đạt chứng chỉ ngôn ngữ Nhật **N5** để mở rộng không gian phát triển.

## 🏆 Hackathons & Projects

- **Devpost Reddit Mod Tools and Migrated Apps Hackathon** — Dự án nghiên cứu phát triển các công cụ tối ưu hóa ứng dụng và tích hợp hệ thống.
- **Các dự án tiêu biểu từng triển khai:**
  - **SmartCenter (.NET 8 Web API):** Hệ thống dịch vụ quản lý và chấm điểm thi tự động tích hợp Mail Notification Service để tự động gửi thông báo điểm qua email.
  - **IoT Smart Parking System:** Giải pháp bãi đỗ xe thông minh toàn diện sử dụng ESP32 và Arduino kết hợp nền tảng Blynk để kiểm soát cổng tự động.
  - **Charity Management Blockchain:** Hệ thống mô phỏng quản lý giao dịch từ thiện minh bạch chạy trên Sepolia Testnet bằng Node.js kết hợp lưu trữ MongoDB.
  - **Online Course Registration System:** Ứng dụng quản lý đăng ký môn học trực tuyến xây dựng trên nền tảng Java Web (JSP, Servlet, JSTL, SQL Server).
